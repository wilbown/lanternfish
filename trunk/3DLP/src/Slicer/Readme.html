Freesteel Slice, command line tool

Version 1.5

After installation, these files and directories exist in your chosen installation folder (this is C:\Program Files (x86)\Freesteel Slicer):

  • dist:
    Contains all DLLs and executables. The installation should have added this to your PATH environment, so that you can execute 'slice.exe' from the command line.
  • Freesteel.URL
    Internet shortcut to our website.
  • example_popen.py
    A python script example, showing how you could integrate the executable in your own application using JSON. You should have some familiarity with python. A similar approach should work for any other programming language, as long as you can execute shell commands, read their standard output stream and decode JSON.
  • Readme.html
    This file.
  • Cube10.stl
    An example of an STL file (ASCII format), a cube of size 10mm.
  • testpart.STL
    Another example of an STL file (ASCII format).

Open a command prompt, for example clicking on "Start" and type "cmd" into the search dialog, or "Run" form (depending on your version of MS Windows) Then type:

>slice

or

>slice "C:\Program Files (x86)\Freesteel Slicer\Cube10.stl"

More instructions can be found on the Freesteel website.
